# ***Speecher.js***

## ***How To Use (format to use) :-***
```html
<script src='https://github.com/sancho1952007/Speecher.js/main.js'></script>
<script>
    speak(Language, Volume, Speech Rate, Text To Say)
</script>
```

<br><br>

## ***Example :-***
```html
<script src='https://github.com/sancho1952007/Speecher.js/main.js'></script>
<script>
    speak('en-IN', 1, 1, "Hello World!")
</script>
```
